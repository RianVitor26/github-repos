## Description 🎮
Criação de perfil e sistema de pesquisa e filtro de repositórios usando a API  do GIthub

## Technologies 💻
* [Next.js](https://nextjs.org/)
* [TypeScript](https://www.typescriptlang.org/)
* [PNPM](https://pnpm.io/)

## Installation 🛠️

```bash
$ pnpm install
```

## Running the API ⚡

```bash
# development
$ pnpm dev
```
